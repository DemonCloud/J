## Install

### with Plug 

``Plug 'DemonCloud/J'``

 Restart Vim and command call ``PlugInstall``



### with Git 

``git clone https://github.com/DemonCloud/J.git``

 Then copy the ``colors/J.vim`` file into your Vim ``colors`` file directory



### last setup

 put this in your ``vimrc`` file

``colorscheme J``
